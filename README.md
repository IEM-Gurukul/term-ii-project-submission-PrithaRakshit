# PCCCS495 – Term II Project

## Project Title
Bank Account Management System

---

## Problem Statement (max 150 words)
Manual banking systems are slow and prone to errors. Managing balances, withdrawals, and different account rules manually is inefficient and unreliable.
---

## Target User
- Small cooperative societies
- Students learning banking systems and OOP concepts

---

## Core Features

-- Account creation with unique ID
- Deposit and withdrawal functionality
- Interest calculation for savings accounts
- Overdraft feature for current accounts
- Validation for insufficient balance

## Proposed Architecture Description
The system follows a modular structure:
- Account class as base
- SavingsAccount and CurrentAccount using inheritance
- BankSystem manages all accounts using ArrayList
- Main class provides user interaction
